# 📊 Dashboard de Vendas em Excel

Este projeto apresenta um **Dashboard de Vendas desenvolvido no Excel**, utilizando **Tabelas Dinâmicas, Segmentação de Dados e Gráficos** para facilitar a análise de desempenho comercial.

O objetivo é transformar dados brutos de vendas em **informações visuais e interativas**, permitindo uma análise rápida dos resultados.

## 📈 Análises disponíveis

O dashboard permite visualizar:

### Vendas Mensais
- Comparação das vendas ao longo dos meses
- Análise de quantidade vendida e faturamento

### Vendas por Categoria
Distribuição percentual das vendas entre:
- Calçado
- Vestuário
- Acessórios

### Ranking de Vendedores
Ranking dos vendedores com base no valor total de vendas, permitindo identificar os melhores desempenhos.

---

## 🎛️ Filtros Interativos

O dashboard possui **segmentação de dados**, permitindo filtrar as informações por:

- Ano
- Mês

Assim é possível analisar rapidamente diferentes períodos de vendas.

---

## 🛠️ Ferramentas utilizadas

- Microsoft Excel
- Tabela Dinâmica
- Segmentação de Dados
- Gráficos dinâmicos
- Dashboard interativo

Praticar técnicas de **análise e visualização de dados no Excel**, transformando informações em insights claros para apoio à tomada de decisão.
